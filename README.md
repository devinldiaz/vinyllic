    Vinyl Collection Manager
  </p>

## Overview
Full-stack app for rendering a list of vinyl albums from a database. 

## Tech Stack  
**Frontend:** Angular  
**Styling:** HTML5, CSS  
**Backend:** Node.js + Express  
**Database:** MySQL  
**Other:** REST API, TypeScript, Git, Postman (for API testing)

## Current Features ✨
- Fetches vinyl entries from a MySQL database  
- Displays vinyl records in a clean Angular UI  
- Responsive card layout with album image, title, & artist  
- Organized backend API structure with REST endpoints


## Project Structure
```
vinyllic/
├── backend/ # Node.js + Express server
└── frontend/ # Angular UI
```

## Run Locally
### 1. Clone the repository
```bash
git clone https://github.com/<your-username>/vinyllic.git
cd vinyllic
```

### 2. **Install Dependencies (Client and Server):**
```bash
# Install server dependencies
cd backend
npm install
node server.js
```
```bash
cd ../frontend
npm install
ng serve
```
